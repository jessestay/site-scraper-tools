# Site Scraper Tools

A powerful web scraping tool for Chrome.

## Features

### Free Version
- Basic web scraping
- Simple asset downloading
- Standard compression
- Up to 3 concurrent downloads

### Premium Version
- Advanced selectors
- High-performance compression
- Unlimited concurrent downloads
- Priority support
- Custom configurations

## Premium Features
